DSCI 100 Problem: Predicting usage of a video game server

Research Report Question: Can player age predict mean session time in the players and "sessions" data sets?

Data Description:

The players.csv data set contains data related to player demographics and overall game contribution meanwhile the sessions.csv contains game time-related info on individual game sessions. For this research question the variables Age and hashedEmail in the players data set and the variables hashedEmail, start_time and end_time in the sessions data set will be required for the analysis. Specifically, I will focus on the relationship between player age (the independent variable) and average session time (the dependent variable). The hashedEmail is a unique identifier for each player which is the necessary link required between the two data sets to relate age and session times. The age data column is a continuous variable that is already given in the player data set; however, session length is a continuous variable that will have to be calculated based on start and end times from the sessions data set.

Potential Issues with Data:

  • Some of the players have missing age data values in the age column.
  • There may be outliers or drastically extreme values for session length which may skew the data.
  • Similarly, the distribution of age may be skewed.

Methods & Plan:

To answer the research question, we will be using a linear regression model. We have chosen a linear regression model since it is best suited for continuous response variables such as session length. This model also easily allows for the integration of other factors such as gaming experience. This model assumes a linear relationship between player age and mean session length, and a normal distribution of errors (residuals). Given the data set, a limitation with this model that may be significant is the models sensitivity to outliers which may be present in session lengths. Another limitation is that if the relationship is in fact non-linear, this may weaken the models performance. Finally, age on its own may not be a strong enough predictor so another factor may have to be incorporated. In terms of preparation and analysis, we will begin by merging the two data sets with the hashedEmail being leveraged as an identifier for each player. Next the average session time will be computed for each player using start and end times. The data will then be split into 75% training set and 25% testing sets, which will allow us to train a linear regression model leveraging age as the predictor variable. Finally, we will make predictions on the training dataset and then evaluate the models performance using RMPSE value. We will then visualize the predictions as a straight line over a scatter plot of the player ages and mean session times to analyze the fit of the resulting line.
